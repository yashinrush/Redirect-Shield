Protection Level Configuration

Choose the level of popup and redirect shielding that fits your browsing comfort. Higher levels may disrupt advanced website behaviors.

Basic

Basic Protection

Blocks programmatic popup tabs. Normal redirects and overlays are permitted.

  • ✔ Blocks window.open()
  • ✘ Allows manual overlay redirects
  • ✘ Allows background tab redirects
Balanced

Balanced Protection

Recommended for most users. Blocks unclicked popups and page navigations while keeping standard link operations.

  • ✔ Blocks window.open()
  • ✔ Intercepts unclicked redirects
  • ✘ Permits hidden click layers
Advanced

Advanced Shield

Blocks popups, overlays, ads, and checks play/download buttons. Lock programmatic triggers inside clicked anchors.

  • ✔ Blocks all popup redirects
  • ✔ Removes click-hijack overlays
  • ✔ Flags fake play/download indicators
Maximum

Strict Shield

Strict enforcement. Completely prevents any navigation or popup targeting external domains. Recommended for streaming/download domains.

  • ✔ Strict main-world limits
  • ✔ Block all external clicks
  • ✔ Deceptive link interrupts
