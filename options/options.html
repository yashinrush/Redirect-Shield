Protection Level Configuration

Choose the level of redirect protection that fits your browsing experience.

Low

Basic Shield

Blocks standard JavaScript-triggered popup windows. Ideal for basic usage.

  • ✔ Blocks window.open()
  • ✘ Allows external tab redirects
  • ✘ Allows clickable overlays
Medium

Moderate Shield

Blocks popups and automatic page redirects without user interactions. Keeps same-tab legitimate actions.

  • ✔ Blocks window.open()
  • ✔ Intercepts Location redirects
  • ✔ Changes target="_blank"
  • ✘ Allows hidden overlays
High

Advanced Shield

Recommended settings. Sweeps ad elements, removes z-index transparent hijacking layers, and locks automatic page redirections.

  • ✔ Blocks all popups
  • ✔ Intercepts all Location & History
  • ✔ Removes click-hijack overlays
  • ✔ Suppresses dynamic ads
Extreme

Strict Shield

Strict security. Blocks any and all external navigations. May break some user login integrations.

  • ✔ Blocks all external clicks
  • ✔ Blocks all window processes
  • ✔ Total overlay removal
  • ✔ Strict main-world locks
