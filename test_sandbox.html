Redirect Shield Sandbox

Use this local testing page to verify extension blocking rules and vector intercepts.

1. Setup Extension

  1. Go to chrome://extensions/ in your browser.
  2. Enable Developer Mode (top right switch).
  3. Click Load unpacked and select this workspace folder: c:\Users\ASUS\Desktop\redirect shield.
  4. Make sure Redirect Shield is active.

2. Test window.open() Interception

Trigger a popup window using the standard JavaScript window.open() method. (Blocked under all Protection Levels).

3. Test Page Redirection

Simulate redirecting the current page using location.assign(), location.replace(), or modifying location.href.

4. Test target="_blank" Clicks

Convert external target links to open in the same tab (target="_self") rather than spawining new ad pages. (Blocked in Medium level or higher).

Open example.com in New Tab

5. Test Click Hijacking (Invisible Overlays)

Simulate streaming sites that overlay a 100% viewport fixed invisible transparent div to capture your first click. (Detected and removed in High and Extreme protection levels).

Overlay state: INACTIVE (Spawn it first)
